A quick look at what shipped this week.

## New features

- **Newsletter signups.** Visitors can now subscribe to email updates directly from the site.
- **Advisory page.** A dedicated page highlights advisory work, with a matching entry on the [work history](/work-experience) timeline.
- **Projects showcase page.** Browse the full project catalog on the new [projects](/projects) page, separate from the home page highlights.
- **FAQ page.** Common questions are now answered in one place on the FAQ page.
- **Keyboard shortcuts overlay.** Press `?` anywhere on the site to see every available shortcut. See the full list on the [keyboard shortcuts](/keyboard-shortcuts) page.
- **Post view counter.** Each post now shows how many times it has been read. Details on the [post view counter](/post-view-counter) page.
- **Site-wide view counter.** A total site views counter joins per-post counts so you can see how content is performing at a glance.
- **Newsletter digest controls.** Pick a digest cadence (daily, weekly, monthly), choose your time zone, preview the next send time, and pause or resume delivery whenever you need a break.

## Updates

- **Projects page filters.** The [projects](/projects) page now includes a search box, a tech-stack filter, and status badges (active, archived, in progress) so visitors can narrow down what they're looking for.
- **Comment improvements.** Comments can now be sorted by newest, oldest, or top, and each post shows a comment count badge so you can spot active threads.
- **Bookmark search.** The bookmarks page now has a search box for quickly finding saved items.
- **Combined post meta.** Post headers show reading time, publish date, and category on a single, tidier line.
- **Theme toggle in the command palette.** Switch between light and dark mode straight from the command palette — no need to hunt for the toggle.
- **Unified feed index.** All available feed formats (RSS, Atom, JSON) are now listed in one place, making it easier to subscribe in your reader of choice.
