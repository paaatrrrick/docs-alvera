A short update covering what shipped this week on the personal website.

## New features

- **Search on the projects page.** The [`/projects`](/projects) showcase now has a search box at the top that filters cards by project name, tagline, or description. Search works alongside the existing tech-stack filter, so you can narrow to a stack and then search within it.

## Updates

- **Empty state for no matches.** When a search or filter combination returns no results, the page now shows a clear "No projects match your search." message instead of a blank grid.

## What this means for you

If you maintain the projects list, the [writing guidance on the projects page](/projects#writing-guidance) covers how to phrase `name`, `tagline`, and `description` so projects surface from search. Lead with the words a visitor would actually type — category, problem space, or stack keywords that aren't already tags.
