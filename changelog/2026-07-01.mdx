Here is what shipped this week.

## New features

- **Search on the [projects page](/projects).** The `/projects` showcase now has a search box that matches project name, tagline, and description. Search works alongside the existing tech filter, so you can combine a keyword with a stack tag to narrow results. When nothing matches, the page shows a friendly empty state instead of a blank grid.

  For guidance on writing project entries so they surface well in search, see the [writing guidance](/projects#writing-guidance) section of the projects page docs.
