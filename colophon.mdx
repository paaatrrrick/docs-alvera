## Overview

The colophon page is a short note on how the personal website is put together — the tools, the hosting, and the principles behind it. It lives at [`/colophon`](https://patrickfoster.dev/colophon) and is rendered by the `Colophon` component in `src/components/Colophon.js`.

## What changed

A new `Colophon` route and component were added:

- **Route:** `/colophon`, registered in `src/App.js`.
- **Component:** `src/components/Colophon.js`.
- **Styles:** `src/components/colophon.css`.

The page displays a header (back link, title, "Last updated" date, intro) followed by a list of sections rendered from a local `sections` array.

## Editing the page

To change what the colophon says, edit the constants at the top of `src/components/Colophon.js`:

```js src/components/Colophon.js
const updatedAt = 'June 2026';

const sections = [
  {
    heading: 'Built with',
    items: [
      'React, bootstrapped with Create React App.',
      // ...
    ],
  },
  // ...
];
```

- Bump `updatedAt` whenever the content changes — it powers the "Last updated" line under the title.
- Add, remove, or reorder entries in `sections` to change the rendered sections. Each section needs a unique `heading` (used as the React key) and a list of `items` strings.
- Tweak layout and typography in `src/components/colophon.css`.
