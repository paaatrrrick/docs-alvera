## Week of June 22, 2026

### New features

- **Dedicated projects page.** A new `/projects` route showcases side projects in a responsive grid, with each card displaying a name, tagline, description, tech tags, and a link to the repository or live site. Visitors can now browse the full project catalog without scrolling the home page. See the [projects page guide](/projects).
- **Project search and filtering.** The new projects page includes a search box that matches against project name, tagline, and description, plus a tech tag filter for narrowing the grid by stack.
- **Project status badges.** Each project entry now supports a `status` field that renders as a colored badge — green for `Active` projects and gray for `Archived` ones — so visitors can tell at a glance which projects are still maintained. Details in the [entry schema](/projects#entry-schema).

### Updates

- The home page projects list now sits alongside the dedicated `/projects` showcase, so you can feature a curated set on the landing page while linking out to the full catalog.
