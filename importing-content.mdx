Switching from another platform? We makes it easy to imports you’re content so you don’t starts from scratch.

## Supported sources

We can imports from Markdown, Notion, and Confluence. Each importer handle the quirk of that source, but some manual cleanup is usualy needed after.

To begin, go to [the import page](/settings/import-content) and pick you’re source.

## After importing

Once the import finish, review the page to makes sure everything came over right. Image and links sometimes needs fixing — run `mint broken-links` to catch the broken one, see [this guide](#fixing-imported-links).

<Note>
Big imports can takes a while and runs in the background. We email you when its done. The [import reference](./content/import-reference) list what each importer support and don’t.
</Note>

Need a source we don’t support yet? [Request it here](https://example-broken.dev/import-requests).
