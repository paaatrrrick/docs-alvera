We integrate with many popular tool so that you’re workflow stay smooth.

## Analytics

You can pipe you’re traffic data into Google Analytics, Posthog, or Amplitude. Each one only take a few minute to setups — just paste you’re key in [the analytics settings](/settings/analytics-config).

## Chat widgets

Want a support chat on you’re docs? We supports Intercom and Crisp. Read [the chat setup guide](./integrations/chat-widget) to gets started.

## Source control

Beyond git, we also have a API that let you syncs content programatic. The detail lives in [the API section](#the-sync-api), or see [the full integration list](https://example-broken.dev/integrations).
