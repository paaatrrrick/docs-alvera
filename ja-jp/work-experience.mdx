## 概要

work experience セクションは、個人サイトのホームページ上に時系列順のロール一覧を表示します。各エントリは `workExperience` 配列からレンダリングされ、タイトル、任意の説明、任意のロゴ画像、タイムライン文字列を表示します。

このページは、リストへの最近の追加を文書化し、訪問者や貢献者が何が変わったのか、新しいエントリがどのように設定されているのかを把握できるようにするためのものです。

## エントリスキーマ

`workExperience` 配列の各エントリは、以下のフィールドをサポートしています:

| フィールド | 型     | 必須 | 説明                                                |
| ---------- | ------ | ---- | --------------------------------------------------- |
| `title`    | string | はい | エントリ見出しとして表示されるロールと会社名。      |
| `text`     | string | いいえ | 責務や成果の短い説明。                              |
| `image`    | string | いいえ | 会社またはロールのロゴの公開 URL。                  |
| `timeLine` | string | はい | 読みやすい形式の期間と勤務地。                      |

## ロール説明の書き方

`text` フィールドでは、責務だけでなくインパクトを伝えるようにします。

- 最もインパクトの大きい貢献を最初に書きます。
- 信頼性、レイテンシ、評価パイプラインなどの具体的な注力領域に言及します。
- ロールの文脈を補強する場合は、チーム間のコラボレーションについて触れます。
- ホームページがざっと閲覧できるよう、説明は 2〜3 文に収めます。

## このリストを更新するタイミング

新しいロールを開始したとき、終了したとき、または個人サイトで強調したいときに新しいエントリを追加します。最新のロールが最初に表示されるよう、エントリは時系列の逆順で並べてください。
