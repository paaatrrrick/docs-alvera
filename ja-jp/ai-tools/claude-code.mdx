Claude Code をセットアップして、Mintlify ドキュメントの執筆とメンテナンスをサポートしてもらいましょう。

## 前提条件

- 有効な Claude サブスクリプション(Pro、Max、または API アクセス)

## セットアップ

<Steps>
<Step title="Claude Code をインストールする">
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
</Step>

<Step title="Mintlify スキルをインストールする">
  ドキュメントディレクトリで次を実行します:

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  これにより、Claude Code に Mintlify のコンポーネントリファレンス、執筆規約、
  ワークフローガイダンスが提供されます。
</Step>

<Step title="プロジェクト指示をカスタマイズする">
  プロジェクトルートの `AGENTS.md` を編集して、プロジェクト固有の用語、
  スタイルの好み、コンテンツの範囲を追加します。
</Step>

<Step title="Claude Code で執筆を開始する">
  ```bash
  claude
  ```
</Step>
</Steps>
