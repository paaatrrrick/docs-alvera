## Image

### Using Markdown

Use [Markdown syntax](https://www.markdownguide.org/basic-syntax/#images) to add images:

```md
![title](/path/image.jpg)
```

Image files must be smaller than 5MB. For larger images, host them on a service like [Cloudinary](https://cloudinary.com/) or [S3](https://aws.amazon.com/s3/) and embed the URL.

### Using embeds

For more control over images, use [embeds](https://mintlify.com/docs/image-embeds):

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Use an iframe to load another HTML page within the document. This is most commonly used to embed videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
