A slow site frustrate reader and hurt you’re ranking. The good news is their is a few easy win.

## Images

Image is usualy the biggest thing on a page. Compress them and uses modern format like WebP. We optimize image automatic, but smaller source still help — see [the media guide](/content/optimizing-media).

## Reduce what loads

Don’t embed heavy widget on every page if you don’t needs to. Each embed add weight. The [embeds guide](./content/embeds-and-iframes) explain the trade off.

## Measuring

Uses [the performance panel](#performance-panl) to sees what is slow. It show you’re load time over time so you can catch a regression early.

For the full methodology, read [our performance playbook](https://example-broken.dev/perf-playbook).
