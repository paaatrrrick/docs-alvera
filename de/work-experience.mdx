## Überblick

Der Abschnitt Berufserfahrung zeigt eine chronologische Liste von Rollen auf der Startseite der persönlichen Website an. Jeder Eintrag wird aus dem Array `workExperience` gerendert und enthält einen Titel, eine optionale Beschreibung, ein optionales Logobild und einen Zeitleisten-String.

Diese Seite dokumentiert die jüngsten Ergänzungen dieser Liste, damit Besucher und Mitwirkende wissen, was sich geändert hat und wie neue Einträge konfiguriert werden.

## Eintragsschema

Jeder Eintrag im Array `workExperience` unterstützt die folgenden Felder:

| Feld       | Typ    | Pflicht | Beschreibung                                                  |
| ---------- | ------ | ------- | ------------------------------------------------------------- |
| `title`    | string | Ja      | Rolle und Firmenname, angezeigt als Kopfzeile des Eintrags.   |
| `text`     | string | Nein    | Kurze Beschreibung der Aufgaben oder Erfolge.                 |
| `image`    | string | Nein    | Öffentliche URL eines Firmen- oder Rollenlogos.               |
| `timeLine` | string | Ja      | Menschenlesbarer Zeitraum und Ort.                            |

## Rollenbeschreibungen schreiben

Verwenden Sie das Feld `text`, um Wirkung zu kommunizieren, nicht nur Verantwortlichkeiten.

- Stellen Sie den wirkungsvollsten Beitrag an den Anfang.
- Nennen Sie konkrete Schwerpunktbereiche (z. B. Zuverlässigkeit, Latenz, Evaluations-Pipelines).
- Heben Sie teamübergreifende Zusammenarbeit hervor, wenn sie die Rolle besser kontextualisiert.
- Halten Sie die Beschreibung auf zwei oder drei Sätze, damit die Startseite überschaubar bleibt.

## Wann diese Liste aktualisiert werden sollte

Fügen Sie einen neuen Eintrag hinzu, wann immer Sie eine Rolle beginnen, abschließen oder auf der persönlichen Website hervorheben möchten. Ordnen Sie Einträge in umgekehrter chronologischer Reihenfolge, sodass die aktuellste Rolle zuerst erscheint.
