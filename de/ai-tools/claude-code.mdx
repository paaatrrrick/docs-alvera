Richten Sie Claude Code ein, um beim Schreiben und Pflegen Ihrer Mintlify-Dokumentation zu helfen.

## Voraussetzungen

- Aktives Claude-Abonnement (Pro, Max oder API-Zugang)

## Einrichtung

<Steps>
<Step title="Claude Code installieren">
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
</Step>

<Step title="Mintlify-Skill installieren">
  Führen Sie in Ihrem Docs-Verzeichnis aus:

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  Damit erhält Claude Code Mintlifys Komponentenreferenz, Schreibstandards
  und Workflow-Anleitung.
</Step>

<Step title="Projektanweisungen anpassen">
  Bearbeiten Sie `AGENTS.md` im Stammverzeichnis Ihres Projekts, um projektspezifische Terminologie,
  Stilpräferenzen und Inhaltsgrenzen hinzuzufügen.
</Step>

<Step title="Mit Claude Code zu schreiben beginnen">
  ```bash
  claude
  ```
</Step>
</Steps>
