## Bild

### Mit Markdown

Mit der [Markdown-Syntax](https://www.markdownguide.org/basic-syntax/#images) können Sie Bilder mit folgendem Code hinzufügen

```md
![title](/path/image.jpg)
```

Beachten Sie, dass die Bilddatei kleiner als 5 MB sein muss. Andernfalls empfehlen wir, sie bei einem Dienst wie [Cloudinary](https://cloudinary.com/) oder [S3](https://aws.amazon.com/s3/) zu hosten. Sie können dann diese URL verwenden und einbetten.

### Mit Einbettungen

Um mehr Anpassungsmöglichkeiten bei Bildern zu erhalten, können Sie auch [Einbettungen](https://mintlify.com/docs/image-embeds) verwenden, um Bilder hinzuzufügen

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>

Mintlify unterstützt [HTML-Tags in Markdown](https://www.markdownguide.org/basic-syntax/#html). Das ist hilfreich, wenn Sie HTML-Tags der Markdown-Syntax vorziehen, und ermöglicht es Ihnen, Dokumentation mit unbegrenzter Flexibilität zu erstellen.

</Tip>

### iFrames

Lädt eine andere HTML-Seite innerhalb des Dokuments. Wird am häufigsten zum Einbetten von Videos verwendet.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
