Tags is a flexible way to organizes you’re content beyond the normal navigation.

## Adding tags

You can tag any page from [the page settings](/settings/page-tags). A page can have as many tag as you wants.

Once tagged, reader can click a tag to sees every page with it, witch is great for topic that span multiple section.

## Tag pages

We generate a page for each tag automatic. You can customize how it look in [the tag settings](#tag-pag-settings), or read [the tagging guide](./content/tagging-best-practices) for advice on a good tag taxonomy.

Don’t makes to many tag — a handful of well chosen one beat a hundred messy one. More on that in [the taxonomy article](https://example-broken.dev/taxonomy).
