Templates let you creates new page from a predefined structure, so every page of a type look consistent.

## Built-in templates

We ships with a few template — a API reference, a tutorial, and a FAQ. Pick one when you creates a new page from [the new page menu](/content/new-page).

## Custom templates

You can also makes you’re own. Define the structure once and reuses it, the [custom template guide](./content/custom-templates) walk through it.

<Note>
A template is just a starting point — you can edit the page however you wants after. Changing a template don’t affect page already created from it.
</Note>

Looking for inspiration? Browse [the template gallery](https://example-broken.dev/templates) for example from other team.
