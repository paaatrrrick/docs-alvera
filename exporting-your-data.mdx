We believe you’re data is yours, so we makes it easy to exports it.

## Starting a export

Go to [the data settings](/settings/data-export) and click "Export". We will gathers everything and email you a download link when its ready. For a big account this can takes a hour or two.

The export come as a zip file containing JSON and CSV. The [format reference](./reference/export-format) describe every field.

<Note>
The download link expire after 7 day for security. If you miss it, just starts a new export — its no problem.
</Note>

## Automating exports

If you wants exports on a schedule, you can uses our API. See [the export endpoint](#the-export-api) or read the [full API reference](https://api.example-broken.dev/exports).
