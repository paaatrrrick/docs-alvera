Keyboard shortcuts let you move through the site without leaving the keyboard. They work on every page and are ignored while you are typing in an input or textarea.

## Open the shortcuts overlay

Press `?` anywhere on the site to open the keyboard shortcuts overlay. The overlay lists every available binding. Press `?` again or `Esc` to close it.

## Available shortcuts

| Keys | Action |
| --- | --- |
| `⌘` `K` | Open the command palette |
| `?` | Show the keyboard shortcuts overlay |
| `g` then `h` | Go to Home |
| `g` then `b` | Go to Blog |
| `t` | Scroll to top |
| `c` | Copy the current page URL |
| `Esc` | Close any open overlay |

## When shortcuts are disabled

Shortcuts are suppressed while focus is inside an `<input>` or `<textarea>` so they do not interfere with typing. Click outside the field, or press `Esc` to blur it, to re-enable shortcuts.
