These shortcuts help you navigate the documentation faster. They work on every page.

## Shortcuts

- Press `/` to focus the search bar.
- Press `g` then `h` to go to the home page.
- Use the arrow keys to move between search results.
- Press `Esc` to close search.
- Press `⌘K` (or `Ctrl+K`) to open the command palette, then press `d` to toggle between light and dark mode.
