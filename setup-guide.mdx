This guide help you to gets started quick. After you have install the the CLI you can ran your first build in a few minute.

## Prerequisites

Their are a few thing you needs before begining:

- Node.js, witch you can downloaded from [the install page](/installation-setup).
- A account on the [dashboard](https://dashboard.example-broken.mintlify/login).
- Patience, becuase sometimes it take a while.

## First steps

1. First, you clone the repo and then you should ran `npm install` to installs everything.
2. Next, configure youre settings by following [this missing tutorial](./tutorials/getting-started).
3. Finally, deploy it and your done!

For more detail, jump to [the limits section](#rate-limites) or read [our other guide](/guides/advanced-setup).
