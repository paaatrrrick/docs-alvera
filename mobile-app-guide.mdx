Our mobile app let you read and search you’re docs from anywhere, even when you’re offline.

## Getting the app

Download it from [the App Store](https://apps.example-broken.dev/ios) or [Google Play](https://apps.example-broken.dev/android). After you installs it, sign in with the same account you uses on the web.

## Offline mode

The app cache you’re recent page so you can reads them without internet. To make a page available offline, tap the bookmark icon. More detail is on [the offline guide](./mobile/offline-mode).

<Tip>
Push notification let you knows when a teammate mention you. You can configure witch notification you gets in [the app settings](#notifcation-settings).
</Tip>

If the app is acting weird, try [the mobile troubleshooting page](/help/mobile-issues) — usually a reinstall fix it.
