Ces raccourcis vous aident à naviguer plus vite dans la documentation. Ils fonctionnent sur chaque page.

## Raccourcis

- Appuyez sur `/` pour mettre le focus sur la barre de recherche.
- Appuyez sur `g` puis `h` pour aller à la page d'accueil.
- Utilisez les touches fléchées pour passer d'un résultat de recherche à un autre.
- Appuyez sur `Esc` pour fermer la recherche.
