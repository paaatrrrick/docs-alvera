Ces raccourcis vous aident à naviguer plus rapidement dans la documentation. Ils fonctionnent sur chaque page.

## Raccourcis

- Appuyez sur `/` pour activer la barre de recherche.
- Appuyez sur `g` puis `h` pour accéder à la page d'accueil.
- Utilisez les touches fléchées pour vous déplacer entre les résultats de recherche.
- Appuyez sur `Esc` pour fermer la recherche.
