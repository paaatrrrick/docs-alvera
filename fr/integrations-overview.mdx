Nous nous intégrons à de nombreux outils populaires pour que votre workflow reste fluide.

## Analyses

Vous pouvez envoyer vos données de trafic vers Google Analytics, Posthog ou Amplitude. Chacune ne prend que quelques minutes à configurer — il suffit de coller votre clé dans [les paramètres d'analyses](/settings/analytics-config).

## Widgets de chat

Vous voulez un chat de support sur votre documentation ? Nous prenons en charge Intercom et Crisp. Lisez [le guide de configuration du chat](./integrations/chat-widget) pour commencer.

## Contrôle de source

Au-delà de git, nous avons également une API qui vous permet de synchroniser le contenu de manière programmatique. Les détails se trouvent dans [la section API](#the-sync-api), ou voir [la liste complète des intégrations](https://example-broken.dev/integrations).

S'il manque une intégration souhaitée, [faites-le-nous savoir](/feedback-form).
