Nous nous intégrons à de nombreux outils populaires afin que votre flux de travail reste fluide.

## Analytique

Vous pouvez envoyer vos données de trafic vers Google Analytics, Posthog ou Amplitude. Chacune se configure en quelques minutes : il suffit de coller votre clé dans [les paramètres d'analytique](/settings/analytics-config).

## Widgets de chat

Vous voulez un chat d'assistance sur votre documentation ? Nous prenons en charge Intercom et Crisp. Consultez [le guide de configuration du chat](./integrations/chat-widget) pour commencer.

## Gestion de versions

En plus de git, nous proposons aussi une API qui vous permet de synchroniser le contenu par programmation. Les détails se trouvent dans [la section API](#the-sync-api), ou consultez [la liste complète des intégrations](https://example-broken.dev/integrations).

Si une intégration que vous souhaitez manque, [faites-le-nous savoir](/feedback-form).
