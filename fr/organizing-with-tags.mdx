Les tags sont une façon flexible d'organiser votre contenu au-delà de la navigation classique.

## Ajouter des tags

Vous pouvez taguer n'importe quelle page depuis [les paramètres de page](/settings/page-tags). Une page peut avoir autant de tags que vous le souhaitez.

Une fois tagués, les lecteurs peuvent cliquer sur un tag pour voir toutes les pages qui le portent, ce qui est idéal pour les sujets qui couvrent plusieurs sections.

## Pages de tags

Nous générons automatiquement une page pour chaque tag. Vous pouvez personnaliser son apparence dans [les paramètres des tags](#tag-pag-settings), ou lire [le guide de taggage](./content/tagging-best-practices) pour des conseils sur une bonne taxonomie de tags.

Ne créez pas trop de tags — une poignée bien choisis vaut mieux qu'une centaine en désordre. Plus de détails dans [l'article sur la taxonomie](https://example-broken.dev/taxonomy).
