Configurez Cursor pour vous aider à rédiger et maintenir votre documentation Mintlify.

## Prérequis

- L'éditeur Cursor installé

## Configuration

<Steps>
<Step title="Ouvrir votre répertoire de documentation dans Cursor">
  Ouvrez la racine de votre dépôt de documentation où se trouve `docs.json`.
</Step>

<Step title="Installer la skill Mintlify">
  Dans le terminal intégré, exécutez :

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  Cela fournit à Cursor la référence des composants Mintlify, les normes de rédaction
  et les conseils de workflow.
</Step>

<Step title="Personnaliser les instructions du projet">
  Modifiez `AGENTS.md` à la racine de votre projet pour ajouter une terminologie propre au projet,
  des préférences de style et des limites de contenu.
</Step>

<Step title="Commencer à rédiger avec Cursor">
  Ouvrez un fichier et utilisez les fonctionnalités d'IA de Cursor pour rédiger et éditer la documentation.
</Step>
</Steps>
