Configurez Claude Code pour vous aider à rédiger et maintenir votre documentation Mintlify.

## Prérequis

- Un abonnement Claude actif (Pro, Max ou accès API)

## Configuration

<Steps>
<Step title="Installer Claude Code">
  ```bash
  npm install -g @anthropic-ai/claude-code
  ```
</Step>

<Step title="Installer la skill Mintlify">
  Dans le répertoire de votre documentation, exécutez :

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  Cela fournit à Claude Code la référence des composants Mintlify, les standards
  de rédaction et des conseils sur le workflow.
</Step>

<Step title="Personnaliser les instructions du projet">
  Modifiez `AGENTS.md` à la racine de votre projet pour ajouter la terminologie
  spécifique au projet, les préférences de style et les limites du contenu.
</Step>


<Step title="Commencer à rédiger avec Claude Code">
  ```bash
  claude
  ```
</Step>
</Steps>
