## Vue d'ensemble

La page de colophon est une courte note sur la façon dont le site web personnel est construit : les outils, l'hébergement et les principes qui le sous-tendent. Elle vit à [`/colophon`](https://patrickfoster.dev/colophon) et est rendue par le composant `Colophon` dans `src/components/Colophon.js`.

## Ce qui a changé

Une nouvelle route et un nouveau composant `Colophon` ont été ajoutés :

- **Route :** `/colophon`, enregistrée dans `src/App.js`.
- **Composant :** `src/components/Colophon.js`.
- **Styles :** `src/components/colophon.css`.

La page affiche un en-tête (lien de retour, titre, date de "Dernière mise à jour", introduction) suivi d'une liste de sections rendues à partir d'un tableau local `sections`.

## Structure de la page

Le composant rend trois sections par défaut, chacune adossée à un objet `{ heading, items }` :

## Éditer la page

Pour modifier ce que dit le colophon, éditez les constantes en haut de `src/components/Colophon.js` :

```js src/components/Colophon.js
const updatedAt = 'June 2026';

const sections = [
  {
    heading: 'Built with',
    items: [
      'React, bootstrapped with Create React App.',
      // ...
    ],
  },
  // ...
];
```

- Mettez à jour `updatedAt` chaque fois que le contenu change — il alimente la ligne "Dernière mise à jour" sous le titre.
- Ajoutez, supprimez ou réordonnez les entrées dans `sections` pour changer les sections rendues. Chaque section a besoin d'un `heading` unique (utilisé comme clé React) et d'une liste de chaînes `items`.
- Ajustez la mise en page et la typographie dans `src/components/colophon.css`.
