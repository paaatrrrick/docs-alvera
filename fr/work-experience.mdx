## Aperçu

La section expérience professionnelle affiche une liste chronologique de postes sur la page d'accueil du site personnel. Chaque entrée est générée à partir du tableau `workExperience` et affiche un titre, une description facultative, une image de logo facultative et une chaîne représentant la chronologie.

Cette page documente les ajouts récents à cette liste afin que les visiteurs et les contributeurs sachent ce qui a changé et comment de nouvelles entrées sont configurées.

## Schéma d'entrée

Chaque entrée du tableau `workExperience` prend en charge les champs suivants :

| Champ      | Type   | Obligatoire | Description                                                  |
| ---------- | ------ | ----------- | ------------------------------------------------------------ |
| `title`    | string | Oui         | Nom du poste et de l'entreprise affiché comme en-tête.       |
| `text`     | string | Non         | Brève description des responsabilités ou accomplissements.   |
| `image`    | string | Non         | URL publique d'un logo d'entreprise ou de poste.             |
| `timeLine` | string | Oui         | Chronologie et lieu en format lisible.                       |

## Rédiger les descriptions de poste

Utilisez le champ `text` pour communiquer l'impact, pas seulement les responsabilités.

- Commencez par la contribution à plus fort impact.
- Mentionnez des domaines de focus concrets (par exemple : fiabilité, latence, pipelines d'évaluation).
- Soulignez la collaboration interéquipes lorsqu'elle apporte un contexte au poste.
- Maintenez la description à deux ou trois phrases pour que la page d'accueil reste facilement consultable.

## Quand mettre à jour cette liste

Ajoutez une nouvelle entrée chaque fois que vous commencez, terminez ou souhaitez mettre en avant un poste sur le site personnel. Classez les entrées dans l'ordre chronologique inverse afin que le poste le plus récent apparaisse en premier.
