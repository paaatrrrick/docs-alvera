## Vue d'ensemble

La section expérience professionnelle affiche une liste chronologique des postes sur la page d'accueil du site personnel. Chaque entrée est rendue à partir du tableau `workExperience` et présente un titre, une description facultative, une image de logo facultative et une chaîne de chronologie.

Cette page documente les ajouts récents à cette liste afin que les visiteurs et contributeurs sachent ce qui a changé et comment configurer de nouvelles entrées.

## Schéma d'une entrée

Chaque entrée du tableau `workExperience` prend en charge les champs suivants :

| Champ      | Type   | Obligatoire | Description                                                       |
| ---------- | ------ | ----------- | ----------------------------------------------------------------- |
| `title`    | string | Oui         | Intitulé du poste et nom de l'entreprise affichés en en-tête.     |
| `text`     | string | Non         | Brève description des responsabilités ou des réalisations.        |
| `image`    | string | Non         | URL publique d'un logo d'entreprise ou de poste.                  |
| `timeLine` | string | Oui         | Période et lieu lisibles par un humain.                           |

## Rédiger les descriptions de poste

Utilisez le champ `text` pour communiquer un impact, pas seulement des responsabilités.

- Mettez en avant la contribution à plus fort impact.
- Mentionnez des axes concrets (par exemple, fiabilité, latence, pipelines d'évaluation).
- Soulignez la collaboration interéquipes lorsqu'elle apporte du contexte au poste.
- Limitez la description à deux ou trois phrases pour que la page d'accueil reste rapide à parcourir.

## Quand mettre à jour cette liste

Ajoutez une nouvelle entrée chaque fois que vous commencez ou terminez un poste, ou lorsque vous souhaitez le mettre en avant sur le site personnel. Triez les entrées dans l'ordre chronologique inverse pour que le poste le plus récent apparaisse en premier.
