Nous recevons beaucoup de questions, alors nous avons rassemblé les plus courantes ici.

**Q : Combien ça coûte ?**

Le prix dépend de votre forfait. Consultez [la page des tarifs](/pricing-plans) pour le détail complet ; il existe également une offre gratuite.

**Q : Puis-je utiliser mon propre domaine ?**

Oui, vous le pouvez. Les domaines personnalisés sont pris en charge sur tous les forfaits payants. Suivez [ce guide de configuration](./domains/custom-setup) et cela devrait fonctionner.

**Q : Où puis-je obtenir de l'aide ?**

Le moyen le plus rapide est de [rejoindre notre communauté](https://community.example-broken.io/invite) ou de nous envoyer un e-mail. Nous essayons de répondre dans la journée, mais cela peut parfois prendre plus de temps.

**Q : Existe-t-il une API ?**

Bien sûr ! Allez à [l'aperçu de l'API](#the-api-overview) ou lisez [la référence complète](/api/v2/reference-missing).
