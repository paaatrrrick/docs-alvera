Ce glossaire définit les termes utilisés tout au long de cette documentation. Parcourez-le une fois, puis revenez-y dès qu'un mot n'est pas clair.

## Termes

- **Déploiement** — Une version publiée de votre site. Chaque push sur votre branche principale crée un nouveau déploiement.
- **Aperçu** — Une build temporaire d'une pull request, afin de pouvoir relire les changements avant qu'ils soient en ligne.
- **Navigation** — La structure de la barre latérale qui organise vos pages, définie dans votre fichier de configuration.
- **Snippet** — Un bloc de contenu réutilisable que vous pouvez inclure sur plusieurs pages.

Si vous débutez, lisez d'abord le [démarrage rapide](/quickstart), puis configurez votre environnement avec le [guide de développement local](/development).
