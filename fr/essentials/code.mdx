## Code en ligne

Pour désigner un `mot` ou une `phrase` comme du code, encadrez-le par des backticks (`).

## Blocs de code

Utilisez des [blocs de code encadrés](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en entourant le code de trois backticks, et faites suivre les backticks d'ouverture du langage de programmation de votre extrait pour obtenir la coloration syntaxique. Vous pouvez également indiquer le nom de votre code après le langage de programmation.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
