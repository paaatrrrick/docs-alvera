## Titres

À utiliser pour les en-têtes de section.

```md
## Titres
```

### Sous-titres

À utiliser pour les en-têtes de sous-section.

```md
### Sous-titres
```

<Tip>

Chaque **titre** et **sous-titre** crée une ancre et apparaît également dans la table des matières à droite.

</Tip>

## Mise en forme du texte

Nous prenons en charge la plupart des mises en forme markdown. Ajoutez simplement `**`, `_` ou `~` autour du texte pour le mettre en forme.

| Style         | Comment l'écrire  | Résultat        |
| ------------- | ----------------- | --------------- |
| Gras          | `**gras**`        | **gras**        |
| Italique      | `_italique_`      | _italique_      |
| Barré         | `~barré~`         | ~barré~         |

Vous pouvez les combiner. Par exemple, écrivez `**_gras et italique_**` pour obtenir du texte **_gras et italique_**.

Vous devez utiliser du HTML pour écrire du texte en exposant et en indice. Autrement dit, ajoutez `<sup>` ou `<sub>` autour de votre texte.

| Taille du texte | Comment l'écrire         | Résultat               |
| --------------- | ------------------------ | ---------------------- |
| Exposant        | `<sup>exposant</sup>`    | <sup>exposant</sup>    |
| Indice          | `<sub>indice</sub>`      | <sub>indice</sub>      |

## Lier vers des pages

Vous pouvez ajouter un lien en encadrant le texte avec `[]()`. Vous écririez `[lien vers google](https://google.com)` pour [lien vers google](https://google.com).

Les liens vers des pages de votre documentation doivent être relatifs à la racine. Autrement dit, vous devez inclure le chemin complet du dossier. Par exemple, `[lien vers texte](/writing-content/text)` renvoie vers la page « Text » de notre section composants.

Les liens relatifs comme `[lien vers texte](../text)` s'ouvriront plus lentement car nous ne pouvons pas les optimiser aussi facilement.

## Citations

### Sur une ligne

Pour créer une citation, ajoutez un `>` devant un paragraphe.

> Dorothy le suivit à travers les nombreuses belles pièces de son château.

```md
> Dorothy le suivit à travers les nombreuses belles pièces de son château.
```

### Sur plusieurs lignes

> Dorothy le suivit à travers les nombreuses belles pièces de son château.
>
> La Sorcière lui ordonna de nettoyer les marmites et les bouilloires, de balayer le sol et d'entretenir le feu avec du bois.

```md
> Dorothy le suivit à travers les nombreuses belles pièces de son château.
>
> La Sorcière lui ordonna de nettoyer les marmites et les bouilloires, de balayer le sol et d'entretenir le feu avec du bois.
```

### LaTeX

Mintlify prend en charge [LaTeX](https://www.latex-project.org) via le composant Latex.

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
