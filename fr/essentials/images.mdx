## Image

### Avec Markdown

La [syntaxe markdown](https://www.markdownguide.org/basic-syntax/#images) vous permet d'ajouter des images en utilisant le code suivant

```md
![title](/path/image.jpg)
```

Notez que la taille du fichier image doit être inférieure à 5 Mo. Sinon, nous recommandons d'héberger l'image sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Vous pourrez ensuite utiliser cette URL pour l'intégrer.

### Avec les intégrations

Pour plus de personnalisation des images, vous pouvez aussi utiliser des [intégrations](https://mintlify.com/docs/image-embeds) pour ajouter des images

```html
<img height="200" src="/path/image.jpg" />
```

</Tip>

### iFrames

Charge une autre page HTML à l'intérieur du document. Le plus souvent utilisé pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
