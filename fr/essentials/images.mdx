## Image

### En utilisant Markdown

Utilisez la [syntaxe Markdown](https://www.markdownguide.org/basic-syntax/#images) pour ajouter des images :

```md
![titre](/path/image.jpg)
```

Les fichiers image doivent peser moins de 5 Mo. Pour des images plus volumineuses, hébergez-les sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/) et intégrez l'URL.

### En utilisant des intégrations

Pour plus de contrôle sur les images, utilisez des [intégrations](https://mintlify.com/docs/image-embeds) :

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>

Mintlify prend en charge les [balises HTML dans Markdown](https://www.markdownguide.org/basic-syntax/#html). C'est utile si vous préférez les balises HTML à la syntaxe Markdown, et cela vous permet de créer de la documentation avec une flexibilité infinie.

</Tip>

### iFrames

Utilisez une iframe pour charger une autre page HTML dans le document. C'est le plus souvent utilisé pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
