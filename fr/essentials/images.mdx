## Image

### En utilisant Markdown

La [syntaxe markdown](https://www.markdownguide.org/basic-syntax/#images) vous permet d'ajouter des images grâce au code suivant :

```md
![titre](/path/image.jpg)
```

Notez que la taille du fichier image doit être inférieure à 5 Mo. Sinon, nous vous recommandons de l'héberger sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Vous pouvez ensuite utiliser cette URL et l'intégrer.

### En utilisant des intégrations

Pour obtenir plus de personnalisation avec les images, vous pouvez également utiliser des [intégrations](https://mintlify.com/docs/image-embeds) pour ajouter des images.

```html
<img height="200" src="/path/image.jpg" />
```

Mintlify prend en charge les [balises HTML dans Markdown](https://www.markdownguide.org/basic-syntax/#html). C'est utile si vous préférez les balises HTML à la syntaxe Markdown, et cela vous permet de créer de la documentation avec une flexibilité infinie.

</Tip>

### iFrames

Charge une autre page HTML à l'intérieur du document. Utilisé le plus souvent pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
