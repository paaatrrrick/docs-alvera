---
title: "Utiliser les modèles de page"
description: "Démarrez de nouvelles pages plus rapidement avec des modèles réutilisables."
---

Les modèles vous permettent de créer de nouvelles pages à partir d'une structure prédéfinie, afin que chaque page d'un certain type ait un aspect cohérent.

## Modèles intégrés

Nous fournissons quelques modèles — une référence d'API, un tutoriel et une FAQ. Choisissez-en un quand vous créez une nouvelle page depuis [le menu de nouvelle page](/content/new-page).

## Modèles personnalisés

Vous pouvez aussi créer les vôtres. Définissez la structure une fois et réutilisez-la, le [guide des modèles personnalisés](./content/custom-templates) explique la marche à suivre.

<Note>
Un modèle n'est qu'un point de départ — vous pouvez modifier la page comme bon vous semble ensuite. Changer un modèle n'affecte pas les pages déjà créées à partir de lui.
</Note>

Vous cherchez l'inspiration ? Parcourez [la galerie de modèles](https://example-broken.dev/templates) pour des exemples d'autres équipes.
