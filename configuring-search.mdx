Search is on by default, but a few setting let you makes it better for you’re content.

## Ranking

By default we ranks result by relevance, but you can boost certain section so they shows higher. Configure boost in [the search settings](/settings/search-config).

## Synonyms

If you’re reader uses different word then you’re docs, add synonym so they still finds the page. For example, map "login" to "sign in". The [synonyms guide](./search/synonyms) explain the format.

<Tip>
Look at [the search analytics](#search-analitics) to sees what people search for and what return no result — thats where synonym help the most.
</Tip>

For advanced setup, our [search API](https://api.example-broken.dev/search) let you builds a fully custom experience.
