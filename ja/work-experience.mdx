## 概要

職務経歴セクションは、個人サイトのホームページにロールを時系列で表示します。各エントリは `workExperience` 配列からレンダリングされ、タイトル、任意の説明、任意のロゴ画像、そして期間を表す文字列を表示します。

このページは、その一覧への最近の追加内容を記録し、訪問者や貢献者が変更点や新規エントリの構成方法を把握できるようにすることを目的としています。

## エントリのスキーマ

`workExperience` 配列の各エントリは、以下のフィールドをサポートします。

| フィールド | 型 | 必須 | 説明 |
| ---------- | -- | ---- | ---- |
| `title` | string | はい | エントリの見出しに表示されるロール名と会社名。 |
| `text` | string | いいえ | 担当業務や成果の簡単な説明。 |
| `image` | string | いいえ | 会社またはロールのロゴの公開 URL。 |
| `timeLine` | string | はい | 人間が読める期間と所在地。 |

## ロール説明の書き方

`text` フィールドでは、担当業務だけでなくインパクトが伝わる内容を書きましょう。

- 最もインパクトのある貢献から書き始める。
- 具体的なフォーカス領域を明示する（例: 信頼性、レイテンシ、評価パイプライン）。
- チーム横断の協働がロールの文脈を補強する場合は、そのことに触れる。
- ホームページがざっと読める状態を保つため、説明は 2〜3 文に収める。

## この一覧を更新するタイミング

個人サイトでロールが始まったとき、終わったとき、あるいは特に強調したいときに、新しいエントリを追加してください。エントリは時系列の新しい順に並べ、最新のロールが一番上に表示されるようにします。
