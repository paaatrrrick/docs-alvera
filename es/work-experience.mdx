## Resumen

La sección de experiencia laboral muestra una lista cronológica de puestos en la página principal del sitio web personal. Cada entrada se renderiza a partir del array `workExperience` y muestra un título, una descripción opcional, una imagen de logotipo opcional y una cadena de cronología.

Esta página documenta las incorporaciones recientes a esa lista para que visitantes y colaboradores sepan qué ha cambiado y cómo se configuran las nuevas entradas.

## Esquema de cada entrada

Cada entrada del array `workExperience` admite los siguientes campos:

| Campo      | Tipo   | Obligatorio | Descripción                                                  |
| ---------- | ------ | ----------- | ------------------------------------------------------------ |
| `title`    | string | Sí          | Puesto y nombre de la empresa que se muestran como cabecera. |
| `text`     | string | No          | Descripción breve de responsabilidades o logros.             |
| `image`    | string | No          | URL pública del logotipo de la empresa o del puesto.         |
| `timeLine` | string | Sí          | Periodo de tiempo y ubicación en formato legible.            |

## Cómo redactar las descripciones de cada puesto

Usa el campo `text` para comunicar impacto, no solo responsabilidades.

- Empieza por la contribución de mayor impacto.
- Menciona áreas de enfoque concretas (por ejemplo, fiabilidad, latencia, pipelines de evaluación).
- Destaca la colaboración entre equipos cuando aporte contexto al puesto.
- Mantén la descripción en dos o tres frases para que la página principal siga siendo fácil de ojear.

## Cuándo actualizar esta lista

Añade una nueva entrada cada vez que empieces, termines o quieras destacar un puesto en el sitio web personal. Ordena las entradas en orden cronológico inverso para que el puesto más reciente aparezca primero.
