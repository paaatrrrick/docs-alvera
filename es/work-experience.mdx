## Visión general

La sección de experiencia laboral muestra una lista cronológica de los puestos en la página principal del sitio web personal. Cada entrada se renderiza a partir del array `workExperience` y muestra un título, una descripción opcional, una imagen de logo opcional y una cadena de cronología.

Esta página documenta las incorporaciones recientes a esa lista para que los visitantes y colaboradores sepan qué cambió y cómo se configuran las nuevas entradas.

## Esquema de entrada

Cada entrada del array `workExperience` admite los siguientes campos:

| Campo      | Tipo   | Obligatorio | Descripción                                                  |
| ---------- | ------ | ----------- | ------------------------------------------------------------ |
| `title`    | string | Sí          | Nombre del puesto y la empresa mostrado como encabezado.     |
| `text`     | string | No          | Descripción breve de responsabilidades o logros.             |
| `image`    | string | No          | URL pública del logo de la empresa o el puesto.              |
| `timeLine` | string | Sí          | Marco temporal y ubicación legibles.                         |

## Cómo redactar las descripciones de los puestos

Usa el campo `text` para comunicar impacto, no solo responsabilidades.

- Comienza con la contribución de mayor impacto.
- Menciona áreas concretas de enfoque (por ejemplo, fiabilidad, latencia, pipelines de evaluación).
- Destaca la colaboración entre equipos cuando aporte contexto al puesto.
- Mantén la descripción en dos o tres oraciones para que la página principal siga siendo fácil de escanear.

## Cuándo actualizar esta lista

Añade una nueva entrada cuando empieces o termines un puesto, o cuando quieras destacarlo en el sitio web personal. Ordena las entradas en orden cronológico inverso para que el puesto más reciente aparezca primero.
