## Resumen

La sección de experiencia laboral muestra una lista cronológica de puestos en la página de inicio del sitio web personal. Cada entrada se renderiza desde el array `workExperience` y muestra un título, una descripción opcional, una imagen de logotipo opcional y una cadena de cronología.

Esta página documenta las incorporaciones recientes a esa lista para que visitantes y colaboradores sepan qué cambió y cómo se configuran las nuevas entradas.

## Esquema de entrada

Cada entrada del array `workExperience` admite los siguientes campos:

| Campo      | Tipo   | Obligatorio | Descripción                                                  |
| ---------- | ------ | ----------- | ------------------------------------------------------------ |
| `title`    | string | Sí          | Puesto y nombre de la empresa mostrado como cabecera de la entrada. |
| `text`     | string | No          | Breve descripción de las responsabilidades o logros.         |
| `image`    | string | No          | URL pública del logotipo de la empresa o del puesto.         |
| `timeLine` | string | Sí          | Marco temporal y ubicación legibles por humanos.             |

## Cómo redactar descripciones de puestos

Usa el campo `text` para comunicar el impacto, no solo las responsabilidades.

- Empieza con la contribución de mayor impacto.
- Menciona áreas de enfoque concretas (por ejemplo, fiabilidad, latencia, pipelines de evaluación).
- Destaca la colaboración entre equipos cuando aporta contexto al puesto.
- Mantén la descripción en dos o tres frases para que la página de inicio siga siendo fácil de escanear.

## Cuándo actualizar esta lista

Añade una nueva entrada cada vez que empieces, termines o quieras destacar un puesto en el sitio web personal. Ordena las entradas en orden cronológico inverso para que el puesto más reciente aparezca primero.
