## Resumen

El sitio web personal ahora cuenta con una página dedicada `/projects` que muestra los proyectos personales en una cuadrícula responsiva. Cada tarjeta muestra el nombre del proyecto, un eslogan de una línea, una descripción más larga, el stack tecnológico como etiquetas y un enlace al repositorio o sitio en vivo.

Las entradas provienen del array `projects` en `src/constants/projects.js` y se renderizan mediante el componente `Projects` en la ruta `/projects`. La página convive con la página de inicio para que las personas puedan explorar el catálogo completo de proyectos sin desplazarse por la landing.

## Cuándo usar esta página

Añade una entrada a `/projects` cuando quieras destacar un proyecto personal con su propia tarjeta, un eslogan y etiquetas de tecnología. Usa la lista de la página de inicio para los proyectos que quieras mostrar al frente; usa `/projects` para el catálogo completo.

## Esquema de entrada

Cada objeto del array `projects` admite los siguientes campos. Todos son obligatorios.

| Campo         | Tipo     | Descripción                                                                                       |
| ------------- | -------- | ------------------------------------------------------------------------------------------------- |
| `name`        | string   | Nombre del proyecto que se renderiza como encabezado y enlace de la tarjeta.                      |
| `tagline`     | string   | Resumen de una línea que aparece justo debajo del nombre.                                         |
| `description` | string   | Párrafo más largo que explica qué hace el proyecto y a quién va dirigido.                         |
| `tech`        | string[] | Etiquetas del stack renderizadas como una lista horizontal (por ejemplo, `['React', 'Rust']`).    |
| `status`      | string   | Estado del proyecto. Usa `'Active'` para proyectos que sigues manteniendo o `'Archived'` para los que ya no actualizas. Se renderiza como una insignia de color en la cabecera de la tarjeta. |
| `link`        | string   | URL a la que enlaza el encabezado de la tarjeta. Se abre en una pestaña nueva.                    |

## Añadir un proyecto

Abre `src/constants/projects.js` y añade un nuevo objeto al array `projects`. Las entradas se renderizan en el orden del array, así que coloca primero el proyecto más reciente o más relevante.

```js src/constants/projects.js
const projects = [
    {
        name: 'Inkwell',
        tagline: 'A markdown-first note-taking app',
        description: 'A local-first notes app with bidirectional links, a command palette, and end-to-end encrypted sync.',
        status: 'Active',
        tech: ['React', 'Rust', 'SQLite'],
        link: 'https://github.com/example/inkwell',
    },
];

export default projects;
```

<Tip>
Mantén `tagline` lo bastante corto para caber en una sola línea en móvil — apunta a menos de 60 caracteres. Reserva la presentación más larga para `description`.
</Tip>

## Guía de redacción

- **Nombre:** Usa el nombre canónico del proyecto. Evita prefijos con emoji; el diseño de la tarjeta se basa en una jerarquía tipográfica limpia.
- **Eslogan:** Empieza por la categoría o la mecánica principal ("Gestor de marcadores autoalojado", no "Un proyecto que hice").
- **Descripción:** Una o dos oraciones. Indica qué hace y a quién va dirigido.
- **Etiquetas de tecnología:** Enumera las piezas principales del stack, no todas las dependencias. De tres a cinco etiquetas se lee mejor.
- **Estado:** Usa `'Active'` para proyectos que sigues manteniendo. Cambia a `'Archived'` cuando dejes de publicar actualizaciones para que las personas sepan que el proyecto está congelado — la insignia pasa de verde a gris.
- **Enlace:** Prefiere la URL del repositorio para proyectos open source o el sitio en vivo para proyectos hospedados.

## Navegación

La página se registra en `src/App.js` como:

```js src/App.js
<Route exact path='/projects' component={Projects} />
```

Enlaza a la página desde cualquier parte del sitio con `/projects`. La página incluye un enlace "← Back home" en su encabezado para quienes lleguen directamente a la ruta.
