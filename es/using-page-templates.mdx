---
title: 'Usar plantillas de página'
description: 'Empieza nuevas páginas más rápido con plantillas reutilizables.'
---

Las plantillas te permiten crear nuevas páginas a partir de una estructura predefinida, para que todas las páginas de un mismo tipo se vean coherentes.

## Plantillas integradas

Incluimos algunas plantillas: una de referencia de API, una de tutorial y una de preguntas frecuentes. Elige una al crear una página nueva desde [el menú de página nueva](/content/new-page).

## Plantillas personalizadas

También puedes crear las tuyas. Define la estructura una vez y reutilízala; la [guía de plantillas personalizadas](./content/custom-templates) lo explica paso a paso.

<Note>
Una plantilla es solo un punto de partida: puedes editar la página como quieras después. Cambiar una plantilla no afecta a las páginas ya creadas a partir de ella.
</Note>

¿Buscas inspiración? Echa un vistazo a [la galería de plantillas](https://example-broken.dev/templates) para ver ejemplos de otros equipos.
