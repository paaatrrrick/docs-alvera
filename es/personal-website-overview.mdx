## Resumen

El repositorio `personalwebsite` es el sitio personal de Patrick Foster, desplegado en [patrickrfoster.com](https://www.patrickrfoster.com). Es una aplicaciГіn React de una sola pГЎgina, inicializada con Create React App y enrutada con `react-router-dom`.

## Stack

- **Framework:** React 18 (Create React App mediante `react-scripts` 5)
- **Enrutamiento:** `react-router-dom` v4 (`Switch` + `Route`)
- **Hosting:** Netlify (script de build `netlify` con `_redirects` en `public/`)
- **Estilos:** archivos CSS por componente (sin framework de CSS)
