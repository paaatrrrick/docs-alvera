Mintlify convierte un repositorio Git de archivos MDX en un sitio de documentación rápido y con búsqueda. Esta página explica las piezas en movimiento para que sepas qué ocurre bajo el capó.

## Del commit a la página publicada

1. **Tú escribes contenido.** Las páginas se redactan en MDX, que es Markdown más componentes de React como `<Card>`, `<Tabs>` y `<Accordion>`.
2. **Tú configuras la navegación.** `docs.json` define pestañas, grupos, temas y elementos globales como la barra de navegación y el pie de página.
3. **Haces push a GitHub.** La aplicación de GitHub de Mintlify detecta el cambio en tu rama de despliegue.
4. **Mintlify compila el sitio.** El contenido se valida, se comprueban los enlaces y el sitio se compila en activos estáticos.
5. **El sitio se publica.** La nueva compilación se despliega en tu dominio, normalmente en cuestión de segundos.

## Desarrollo local

Usa la CLI de Mintlify para previsualizar los cambios antes de hacer push:

```bash
mint dev
```

Esto sirve tu documentación en `http://localhost:3000` con recarga en caliente. Para detectar problemas pronto, ejecuta:

```bash
mint validate
mint broken-links
```

## Modelo de autoría

- **Archivos MDX** — cada página es un archivo `.mdx` con frontmatter YAML (`title`, `description`).
- **Componentes** — Mintlify incluye componentes integrados para llamadas, pestañas, grupos de código, acordeones, tarjetas y más.
- **Fragmentos** — el contenido reutilizable vive en `snippets/` y se puede importar en cualquier página.
- **Imágenes y activos** — se referencian por ruta relativa desde la raíz del repositorio.

## Configuración

`docs.json` es la única fuente de verdad para la estructura del sitio:

- `navigation` — pestañas, grupos y orden de páginas.
- `colors`, `logo`, `favicon` — branding.
- `navbar`, `footer` — enlaces globales y redes sociales.
- `contextual` — controla el menú "Copiar / Abrir en ChatGPT" en cada página.

## Búsqueda e IA

Cada sitio publicado incluye:

- **Búsqueda de texto completo** indexada automáticamente en cada compilación.
- **Chat con IA** que responde a preguntas usando tu documentación como contexto.
- **Integraciones de MCP y LLM** para que herramientas como Cursor y Claude puedan hacer referencia a tu documentación.
