Configura Cursor para que te ayude a escribir y mantener tu documentación de Mintlify.

## Requisitos previos

- Editor Cursor instalado

## Configuración

<Steps>
<Step title="Abre tu directorio de documentación en Cursor">
  Abre la raíz del repositorio de documentación donde se encuentra `docs.json`.
</Step>

<Step title="Instala la skill de Mintlify">
  En la terminal integrada, ejecuta:

  ```bash
  npx skills add https://mintlify.com/docs
  ```

  Esto proporciona a Cursor la referencia de componentes, los estándares de escritura
  y las pautas de flujo de trabajo de Mintlify.
</Step>

<Step title="Personaliza las instrucciones del proyecto">
  Edita `AGENTS.md` en la raíz de tu proyecto para añadir terminología específica del proyecto,
  preferencias de estilo y límites del contenido.
</Step>

<Step title="Empieza a escribir con Cursor">
  Abre un archivo y utiliza las funciones de IA de Cursor para redactar y editar la documentación.
</Step>
</Steps>
