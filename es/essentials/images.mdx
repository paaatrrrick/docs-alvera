## Imagen

### Usando Markdown

Usa la [sintaxis Markdown](https://www.markdownguide.org/basic-syntax/#images) para añadir imágenes:

```md
![title](/path/image.jpg)
```

Los archivos de imagen deben pesar menos de 5 MB. Para imágenes más grandes, alójalas en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/) e incrusta la URL.

### Usando incrustaciones

Para tener más control sobre las imágenes, usa [incrustaciones](https://mintlify.com/docs/image-embeds):

```html
<img height="200" src="/path/image.jpg" />
```

Mintlify admite [etiquetas HTML en Markdown](https://www.markdownguide.org/basic-syntax/#html). Esto es útil si prefieres las etiquetas HTML a la sintaxis Markdown, y te permite crear documentación con una flexibilidad infinita.

</Tip>

### iFrames

Usa un iframe para cargar otra página HTML dentro del documento. Es lo más habitual para incrustar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
