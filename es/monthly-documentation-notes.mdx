Aquí tienes un breve resumen de los cambios que hicimos en la documentación este mes. Nada importante, solo mejoras constantes.

## Aspectos destacados

- Aclaramos la [guía de inicio rápido](/quickstart) para que la primera compilación sea más fácil de seguir.
- Añadimos algunas páginas de referencia pequeñas, incluido un glosario.
- Ordenamos la navegación para que las páginas relacionadas queden más cerca entre sí.

Publicamos una nota como esta cada mes. Si tienes comentarios sobre la documentación, nos encantaría conocerlos.
