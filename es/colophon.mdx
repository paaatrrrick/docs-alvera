## Resumen

La página de colofón es una breve nota sobre cómo está construido el sitio web personal: las herramientas, el alojamiento y los principios detrás de él. Vive en [`/colophon`](https://patrickfoster.dev/colophon) y la renderiza el componente `Colophon` en `src/components/Colophon.js`.

## Qué cambió

Se añadieron una nueva ruta y componente `Colophon`:

- **Ruta:** `/colophon`, registrada en `src/App.js`.
- **Componente:** `src/components/Colophon.js`.
- **Estilos:** `src/components/colophon.css`.

La página muestra un encabezado (enlace de regreso, título, fecha de "Última actualización", introducción) seguido de una lista de secciones renderizadas a partir de un arreglo local `sections`.

## Estructura de la página

El componente renderiza tres secciones por defecto, cada una respaldada por un objeto `{ heading, items }`:

## Editar la página

Para cambiar lo que dice el colofón, edita las constantes en la parte superior de `src/components/Colophon.js`:

```js src/components/Colophon.js
const updatedAt = 'June 2026';

const sections = [
  {
    heading: 'Built with',
    items: [
      'React, bootstrapped with Create React App.',
      // ...
    ],
  },
  // ...
];
```

- Actualiza `updatedAt` cada vez que cambie el contenido: alimenta la línea "Última actualización" debajo del título.
- Añade, elimina o reordena entradas en `sections` para cambiar las secciones renderizadas. Cada sección necesita un `heading` único (usado como la clave de React) y una lista de cadenas `items`.
- Ajusta el diseño y la tipografía en `src/components/colophon.css`.
